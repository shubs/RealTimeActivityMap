# What is this ?
This is a map traking events
![Real time traking map](http://sharma.fr/screen.png "Real time traking map")

# How to run it

```
$> node server.js
```
And visit http://localhost:3000

# The flow
The server sends data to the front using socket io

# Dependencies
- Mapbox.js
- Socket.io
- Express.js
